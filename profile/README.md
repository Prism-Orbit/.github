## Featured Work

### 🔵 SK Jalrakshak — IoT Energy Monitoring Platform

> An enterprise-grade platform for real-time electrical monitoring, remote relay control, and instant push alarms — built for industrial deployments.

**Stack:**  
`ESP32` `C++ (Arduino)` `MQTT` `Node.js` `TypeScript` `React Native Web` `Flutter` `PostgreSQL (Supabase)` `MongoDB Atlas` `Socket.io` `Firebase`

**Architecture highlights:**
- Dual-database hybrid engine — PostgreSQL for relational data, MongoDB time-series for high-volume telemetry
- Dynamic MQTT connection pool manager with AES-256-CBC encrypted credentials
- Row Level Security (RLS) via JWT custom claims on Supabase
- Flutter Riverpod state management with native Keystore/Keychain encrypted token storage
- Python hardware simulator for full end-to-end testing without physical hardware

→ [View the showcase repo](https://github.com/prism-orbit/sk-jalrakshak)
